## `javac.exe`
* 자바 컴파일러
* 사람이 작성한 문장을 기계어로 번역
* 소스 파일(*.java)을 클래스 파일(*.class)로 변환

## `java.exe`
* 자바 인터프리터
* 자바 프로그램(클래스 파일)을 실행

## `클래스`
* 자바 프로그램의 단위
* 자바 프로그램은 클래스들로 구성

```java
class 클래스이름{ //클래스의 시작
    /*모든 문장들은 클래스의 {}안에 있어야 한다*/
} //클래스의 끝
```

## `main메서드`
* 자바 프로그램의 시작점
* 이 메서드 없이 실행불가

```java
class 클래스 이름{
    public static void main(String[] args){ //mian메서드의 시작
    /* 실행할 문장을 넣는다.(첫 문장부터 순서대로 실행됨) */
    } //mian메서드의 끝
}
```
